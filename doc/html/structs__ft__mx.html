Detailed Description

Base for typedef t_ft_mx.

This type is used to store a linearly-storaged (row-major order) matrix of double type elements.

Field Documentation

◆ d

s_ft_mx::d

The elements, stored linearly using a row-major order. Element addressing follows this syntax: data[i * n + j] (where i represents the row, m the number of rows and j the column).
