◆ DEF_DRW_HEIGHT

#define DEF_DRW_HEIGHT   500

The default drawing height at startup. This also determines the GUI's window height through DEF_WLAYOUT_RATIO.

◆ DEF_DRW_WIDTH

#define DEF_DRW_WIDTH   500

The default drawing width at startup. This also determines the GUI's window width through DEF_WLAYOUT_RATIO.

◆ DEF_WLAYOUT_RATIO

#define DEF_WLAYOUT_RATIO   1.05

The ratio to calculate the starting GUI's window size from DEF_DRW_WIDTH and DEF_DRW_HEIGHT. After a window resize, the inverse operation is done: the new window size detemines the drawing's width and height using this constant.

◆ fdf_start_gui()

int fdf_start_gui ( t_fdf *  fdf)

fdf_start_gui – TODO.

This is the main function of the program. Called by main after a successful parsing of the map and initialization of the program's central data structure (t_fdf fdf). It is responsible for every initialization of the GUI's elements (window, image) and the registration of the event hooks. It calls the mlx_loop() function which starts everything else, most importantly, the first rendering via the generic hook fdf_generichook() on the first loop. Finally, once out of the loop (after mlx_close_window() is called somewhere else TODO) this function calls the mlx_terminate() function, freeing the MLX42's resources and returns to main() where extra cleaning is done before terminating the program.

[in,out]fdf- The fdf structure.

Non-zero value if correct. A value of 0 is returned if error.
