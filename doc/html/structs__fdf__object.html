Detailed Description

Base for typedef t_fdf_object.

This type is used to store a 3D object. That is: its vertex matrix for each of the spaces (model, world, camera, and screen); its edges; and its triangles. This type also contains the auxiliary parameters for vertex transformations to go from one space to the next such as rotations, translations, scaling, and 2D projection.
