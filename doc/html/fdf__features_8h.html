Detailed Description

Header file for functions to respond to GUI events that are supposed to offer a feature to the user (rotate, translate, zoom, ...).

The GUI hooks just call these functions which take the required actions and re-renders.

Go to the GUI hooks functions (both key and generic hooks) to edit the events (such as key presses) that match the features in this file.
